Mapping by parsing

In some scenarios we have to parse input objects field to map it target's field.
Forexample, let's see this two objects 

class PersonDTO{
  private String dateOfBirth;
	private String name;
	...
	}

and 

	  class Person{
  		private String name;
	  	private String year;
	  	private String month;
	  	private String day;
		...
		}

In this case, PersonDTO's dateOfBirth's value format is yyyy-MM-dd
And we should parse this in order to map Person class's fields (year, month, day).
