Clear array

Postgresql already have array_remove function to remove elements from array, but this doesn't clears array, it removes elements that only equals to target.
For exmaple, 

SELECT array_remove(ARRAY[1,2,3,2], 2);

returns {1,3} To clear array e.i. remove all of elements we can do this trick in pgpsql: lets suppose array is 

arr integer[];

to clear this array, just do: 

arr = '{}'::integer[];
