Fixing Forbidden error in apache server

If you configure apache2 server on ubuntu for custom directory. Let's say /home/user/my-site.
You highly likely get Forbidden error during accessing to your localhost.
To fix this situation add following script to your /etc/apache2/apache2.conf 

<Directory /home/user/my-site>
    Options Indexes FollowSymLinks
    AllowOverride None
    Require all granted
</Directory>
